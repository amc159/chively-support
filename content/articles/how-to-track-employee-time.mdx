## Overview

This guide covers How to Track Employee Time in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. Set up the time attendance feature on the web management system.
2. A. Setting up the time attendance feature on the Web management system.
3. 2. From the navigation menu, click Timekeeping and then select Active.
4. 3. Select the Settings tab. The program has a default. If the restaurant's workflow differs from the information here, the manager can reconfigure it.
5. Select the time range, then click Load data.
6. The program displays the total working hours and detailed timekeeping information for each employee for each day, as shown in the image below.
7. Note: You are only allowed to view the timesheet for a maximum of one month (31 days). If the restaurant selects a longer period, a warning message will appear on the screen: " Your date range cannot exceed a month. Try.
8. Select the employee row you want to edit and click Edit to make adjustments if there are any discrepancies.
9. Click Add row/Delete row to add or delete Clock in/Clock out timers.
10. Enter the reason for modifying Clock in/Clock out.
11. Case 1: The restaurant has set up a "Remind employees to clock in" feature in the Timekeeping/Settings section of the management page.
12. On the screen displaying the message "Clock in," press "Clock in" to confirm clocking in.
