## Overview

This guide covers How To Activate Your Devices in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. Log in to the POS system with your Username and Password, then the system will display this warning:.
2. Using the menu, navigate to the Back-End of the login page, then follow these steps:. Select Subscription/Devices. Find the name of the device you tried to log in with and select Activate.
3. Now that activation is complete, you’re ready to start using Chively software on your device!
4. Enter the new name in the Device name line.
5. Select the device name and press Delete.
6. A confirmation screen will appear, then select OK to delete the device.
