## Overview

This guide covers Setting Up Label Printing on the Management Page in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. To add a new label template, click Add printer template and select Label printer, then click OK.
2. Enter the information that you want to be visible on the label.
3. If you want to add more information, click Add Row. Then fill in the title and content.
4. You have now successfully set up label printing on the management page of the Chively software!
