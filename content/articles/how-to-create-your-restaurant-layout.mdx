## Overview

This guide covers How To Create Your Restaurant Layout in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. Establish your restaurant area on the Management page.
2. In the Layout Settings section on the start-up interface, select Restaurant layout settings.
3. Click Edit and select Add Area, then add a name for that area. You can now drag and drop tables into the named area.
4. To edit the table names, number of seats, and table type, click on the table you want to edit. Fill in the information of table name (Table name), number of seats (No. of seats), and select the table type (Table.
5. Enter the number of tables corresponding to each area (Number of tables).
6. Choose the table model (Round or Square) and the number of seats at the table.
7. Fill in the table row number.
8. Fill in the prefix (Prefix) and the starting number (Start from)Then, press OK.
9. Method 2: Use the mouse to drag and drop table icons to change positions on the diagram.
10. You can also choose a background image for the diagram by clicking "Select background" and then uploading your chosen image. If you want to remove a background image, click Remove background.
11. Click the icon to edit the table layout.
