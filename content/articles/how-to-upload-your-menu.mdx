## Overview

This guide covers How to Upload Your Menu in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. On the start-up interface, select Add item manuallyStep 2: Click AddSee more detailed instructions for adding specific items:Add DishAdd Combo.
2. Instructions to import a menu from an Excel file:Step 1: In the Menu section on the start-up interface, select Import from Excel file.

3. Download the Excel menu file and, if needed, download an Excel template file.
4. Copy the restaurant menu list into the downloaded Excel template.
5. Save and close the Excel fileStep 5: In the Menu Declaration section on the start-up interface, select Import from ExcelStep 6: Upload the imported Excel file to the Chively software.
6. Note: If the Excel file has a duplicate item code with an item already in the system, do one of the following:.
7. Select Update if you want to update the information on the Excel file of the item with the duplicate code in the software.
8. Select Skip if you want to keep the information of the item with the same code on the software, do not update from the Excel file.
9. Click ContinueStep 8: Check how many valid and invalid items have been added to the software.
10. Click Execute, then the system will import the Menu items into the software and notify you of the results.
11. Click Close and you’ve now successfully imported your Menu into the Chively system!
