## Overview

This guide covers Setting Up Automatic Promotions in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. On the management page, go to **Promotion**.
2. Click **Add** to create a new promotion.
3. Enter the promotion name.
4. Set the application period, date(s), and time(s).
5. Choose the discount type (percentage or fixed amount) and enter the value.
6. Enable **Automatically apply this promotion program upon checking out** if you want auto-apply at checkout.
7. Click **Save**.
