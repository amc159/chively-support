## Overview

This guide covers Setting Up Automatic Promotions in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. 1. On the admin page2. On POS.
2. 1. On the admin page.
3. Go to Promotion, then click Add.
4. Enter the name of the promotion, select the application period, and check the application date(s) and application time(s) to ensure it’s accurate.
5. Choose a discount by percentage or amount. Check the box labeled "Automatically apply this promotion program upon checking out" to make the promotion automatic for all transactions.
