## Overview

This guide covers How To Enable Two-factor Authentication in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. Select the Authentication via Google Authenticator app option, then click Continue.
2. You can download the Google Authenticator app from the App Store (IOS) and Google Play Store (Android).
3. Note: If the message Invalid code appears, get a new code from the Google Authenticator app.
4. Select the Authentication via Email option, then click Continue.
5. Enter your email address and select Send Code. Check your email for the code, enter the code you’ve received, and then select Continue.
6. After setup is complete, you can reconfigure the settings. Select your username on the home screen, then click Two-Factor Authentication.
7. Reenter your password, then select Continue.
8. You can then select Delete authentication method or Delete email.
9. Click Yes to confirm the deletion of 2FA authentication or the email address.

- Confirm the change is visible in the expected Chively screen.
- Run a quick real-world test to ensure behavior is correct.
- If results are unexpected, double-check permissions and device/network status.
