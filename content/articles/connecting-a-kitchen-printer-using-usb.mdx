## Overview

This guide covers Connecting a Kitchen Printer Using USB in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. Go to Settings.
2. On Printer & Template, scroll down to Printer & Label ticket and click Settings.
3. Select a printer to connect via USB and click Save.
4. A confirmation prompt will appear. Click OK to proceed.
5. Note: The printer is now connected and will then display the confirmation.
