## Overview

This guide covers How to Set Up an Administrator Security Code for Unauthorized Employees in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. 1. From the Management Page.
2. On the management page, navigate to System Settings, then Role Management. Select the role you want to assign permissions to.
3. From the Order tab, select Show unauthorized functions.
4. Do the same for the Payment, Reporting, and Orders tabs to assign these permissions to employees. Then, click Save to save the settings.
5. If the cashier wants to cancel an order after it’s been sent to the kitchen, they need to click Cancel Item. Then, the program will display a screen to enter the security code.
6. You have now successfully added the administrator code function for unauthorized employees in Chively!
