## Overview

This guide covers How to Set Up an Administrator Security Code for Unauthorized Employees in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. On the management page, go to **System settings -> Role management**.
2. Select the role that needs controlled access.
3. In the **Order** tab, enable **Show unauthorized functions**.
4. Repeat this setting for **Payment**, **Reporting**, and **Orders** as needed.
5. Click **Save**.
6. On POS, test a restricted action (for example, canceling a sent item) and confirm the admin security-code prompt appears.
