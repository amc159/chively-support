## Overview

This guide covers How to Set Up Printers for Cashiers and Kitchen Staff in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. On the home screen, select Settings.
2. Select Printer & Template.
3. In the Printer settings, select the printer you want to use for Cashiers.
4. Click Print test to check your invoice template or to make changes as necessary.
