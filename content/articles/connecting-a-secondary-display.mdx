## Overview

This guide covers Connecting a Secondary Display in Chively using a consistent, support-friendly workflow.

## Before you begin

- Confirm you can access the Chively Management Page or POS as needed.
- Make sure your account has permission to change these settings.
- Keep a test order, test payment, or test user available for validation.

## Steps

1. Connect a Secondary Display.
2. On the main screen, select Settings.
3. Select Second screen and check Allow to use a second screen. Select Slide show or Video to display content on the secondary screen.
4. Update the image or video according to the selection above.

> Add screenshots for each major step before publishing.

## Verification

- Confirm the change is visible in the expected Chively screen.
- Run a quick real-world test to ensure behavior is correct.
- If results are unexpected, double-check permissions and device/network status.
