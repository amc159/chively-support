# Sanity Editor Guide for Copy Editors

This guide explains how to manage Chively help content in the browser.

## Access

1. Open the website URL and go to `/studio`.
2. Sign in with your invited Sanity account.
3. Use the left navigation to open **Articles** or **Categories**.

## Create a New Article

1. Open **Articles** and click **Create new**.
2. Fill in:
   - `Title`
   - `Slug` (Generate from title, then verify)
   - `Short Description`
   - `Primary Category`
   - `Tags`
   - `Status` (`draft`, `beta`, or `published`)
   - `Published At` and `Updated At`
3. Write the content in **Article Body**.
4. Save and publish.

## Add Screenshots/Images

1. In **Article Body**, insert an image block.
2. Upload the screenshot.
3. Add required `Alt text`.
4. Add optional `Caption` if needed.
5. Save and publish updates.

## Assign Categories

- Use **Primary Category** for the main placement.
- Use **Secondary Categories** if the article also fits other sections.

## Content Quality Checklist

- Keep steps clear and numbered.
- Use consistent terminology (`POS`, `Management Page`, `KDS`).
- Verify dates and tags are correct.
- Keep screenshot captions concise and helpful.

## Draft vs Published

- Use `draft` while writing.
- Use `beta` for review-ready content that still needs QA.
- Use `published` when final and approved.
